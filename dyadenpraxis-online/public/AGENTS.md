# public

## Purpose
Static assets served at the web root — favicon and service worker for push notifications.

## Key Files

| File | Description |
|------|-------------|
| `favicon.ico` | Application favicon |
| `sw.js` | Service worker for web push notification handling |

## For AI Agents

### Working In This Directory
- Files here are served as-is (no build processing)
- `sw.js` must remain vanilla JavaScript (no TypeScript, no imports)
- Changes to `sw.js` affect push notification behavior for all users
